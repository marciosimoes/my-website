A slotcar controlled via web app with Arduino

During my couse of IT for Web Development at ETEC Pedro Ferreira Alves, a project called "WebRacer" was carried out that aimed to control a slotcar via a web application.

The project was built using an Arduino and a web interface was developed to control the cars in the game.

Technologies used: Arduno, PHP, MySQL, Apache Server.
