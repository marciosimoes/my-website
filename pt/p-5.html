Um autorama controlado via web app com Arduino

Durante o meu curso de Informática para Internet na ETEC Pedro Ferreira Alves, foi realizado um projeto denominado "WebRacer" que visava controlar um autorama através de uma aplicação web.

O projeto foi construído usando um Arduino e uma interface web foi desenvolvida para controlar os carros no jogo.

Tecnologias usadas: Arduno, PHP, MySQL, Apache Server.
